# Structural Design Patterns

Structural patterns explain how to assemble objects and classes into larger structures.

## Patterns Included

- **Adapter**: `Adapter.cpp`
- **Bridge**: `Bridge/`
- **Composite**: `Composite/`
- **Decorator**: `Decorator/`
- **Facade**: `Facade.cpp`
- **Flyweight**: `Flyweight/`
- **Proxy**: `Proxy/`

Each file or directory contains a self-contained example of the pattern.
