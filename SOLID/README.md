# SOLID Design Principles

SOLID is a mnemonic for five design principles intended to make software designs more understandable, flexible, and maintainable.

## Principles

- **S** - Single Responsibility Principle
- **O** - Open/Closed Principle: `Open-Close.cpp`
- **L** - Liskov Substitution Principle: `LiskovSubstitution.cpp`
- **I** - Interface Segregation Principle: `InterfaceSegregation.cpp`
- **D** - Dependency Inversion Principle: `DependencyInversion.cpp`
