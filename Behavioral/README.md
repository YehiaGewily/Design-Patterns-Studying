# Behavioral Design Patterns

Behavioral patterns are concerned with algorithms and the assignment of responsibilities between objects.

## Patterns Included

- **Chain of Responsibility**: `Chain/`
- **Command**: `Command.cpp`
- **Interpreter**: `Interpreter_Lexical&Parsing.cpp`
- **Iterator**: `Iterator/`
- **Mediator**: `Mediator/`
- **Memento**: `Memento.cpp`
- **Observer**: `Observer/`
- **State**: `State/`
- **Strategy**: `Strategy.cpp`
- **Visitor**: `ClassicVisitor.cpp`, `AcyclicVisitor.cpp`
- **Multimethods**: `Multimethods.cpp`

Each file or directory contains a self-contained example of the pattern.
